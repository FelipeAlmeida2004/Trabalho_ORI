# Trabalho_ORI

### Objetivos:

#### [ ] Ler o arquivo de Pesquisadores
#### [ ] Criar um grafo de Colaboradores dos pesquisadores
#### [ ] Criar tabela Hash para busca de nomes.
#### [ ] Criar uma tabela Hash para detectar se um trabalho é o mesmo para dois pesquisadores.
#### [ ] Tratar Colisão Tradicional: dois trabalhos diferentes colidem na tabela
#### [ ] Tratar Colisão Repetida: o mesmo trabalho é adicionado mais de uma vez na Hash

#### Esolher umas das seguintes operações:
##### - [ ] Dado um nome, listar os nomes dos seus colaboradores (com quem ele está conectado no grafo).
##### - [ ] Dado o título de um trabalho, listar os nomes de todos os autores.
##### - [ ] Calcular o maior grau de um vértice do grafo.
##### - [ ] Calcular o grau médio do grafo.
